# Telegram → WordPress Publisher Bot

Бот для публикации статей на [инвестиционно.рф](https://xn--b1aghca0aeahc1am7c.xn--p1ai) из Telegram.

## Что делает

1. Принимает ссылку на новость или текст
2. Парсит и рерайтит через LLM (Moonshot) в человеческом стиле
3. Автоматически подбирает категории из всех таксономий WordPress
4. Показывает полный текст + объем + определенную таксономию
5. Позволяет редактировать текст и категории перед публикацией
6. Публикует на сайт в правильный Custom Post Type (новости, события, аналитика...)

## Стек

- Python 3.11 + aiogram 3.x
- SQLite (черновики, кэш таксономии)
- Moonshot API (рерайт + категоризация)
- Railway (деплой)

## Деплой на Railway

1. Залейте код в GitHub-репозиторий
2. [Railway](https://railway.app) → New Project → Deploy from GitHub repo
3. Variables → добавьте все из `.env.example`
4. Volume → Name: `bot-data`, Mount: `/app/data`
5. Готово — деплой автоматический

## Переменные окружения

| Variable | Описание | Обязательно |
|----------|----------|-------------|
| `TG_BOT_TOKEN` | Токен от @BotFather | Да |
| `ALLOWED_USER_IDS` | Telegram user ID (через @userinfobot) | Да |
| `WP_BASE_URL` | URL WordPress-сайта | Да |
| `WP_LOGIN` | Логин администратора WP | Да |
| `WP_PASSWORD` | Application Password из WP | Да |
| `MOONSHOT_API_KEY` | API ключ Moonshot/Kimi | Да |
| `MOONSHOT_BASE_URL` | `https://api.moonshot.ai/v1` | Да |
| `MOONSHOT_MODEL` | `moonshot-v1-128k` | Да |
| `RAILWAY_RUN_UID` | `0` (для записи в volume) | Да |

## Команды бота

- `/start` — начать
- `/cancel` — отменить текущую операцию
- `/sync` — обновить кэш таксономий с сайта (автообнаружение новых)

## Поток работы

```
Ссылка/текст → Парсинг → Рерайт (человеческий стиль) → Категоризация LLM
→ 📊 Объем текста → 🤖 Таксономия LLM → Полный текст + кнопки
→ [✅ Опубликовать / 📝 Черновик / ✏️ Текст / 🏷 Категории / 🔄 Заново / ❌ Отмена]
→ Публикация в правильный CPT (novost/sobytie/analitika/...)
```

## Публикация vs Черновик

| Кнопка | Результат |
|--------|-----------|
| ✅ **Опубликовать** | Сразу на сайте, видно всем |
| 📝 **Черновик** | Сохранено в WordPress как `draft`, видно только в админке. Можно доработать вручную. |

## Редактирование перед публикацией

**Текст:**
- ✏️ **Редактировать текст** → 📰 Заголовок / 📝 Контент / 📄 Excerpt
- При редактировании показывается полный текст
- Кнопка ↩️ Назад на каждом шаге
- Можно писать правки прямо в чат (LLM применит автоматически)

**Категории:**
- 🏷 **Редактировать категории** → выбор таксономии → чекбоксы с пагинацией
- Поддерживаются все таксономии сайта (динамически)

## Человеческий стиль текста

Бот использует 8 техник для естественного звучания:
1. **Burstiness** — варьирование длины предложений
2. **Эмоциональные маркеры** — риторические вопросы, ирония, личная позиция
3. **Разговорный стиль** — замена канцеляризмов
4. **Структурные неровности** — тире, скобки, эллипсис
5. **Идиомы** — "что, согласитесь, немало", "мягко говоря"
6. **Текстовая шероховатость** — естественная пунктуация
7. **Конкретика** — аналогии, сравнения вместо абстракций
8. **Контекст** — исторические параллели, прогнозы с оговорками

## Custom Post Types

Бот автоматически определяет тип записи по категории:

| Категория | Тип записи |
|-----------|-----------|
| Новости | `novost` |
| Событие | `sobytie` |
| Аналитика | `analitika` |
| Интересное | `interesnoe` |
| Компания | `kompaniya-p` |
| Тренды | `trends` |

## Добавление новой таксономии

1. Создайте таксономию в WordPress
2. В боте отправьте `/sync`
3. Бот автоматически обнаружит и подхватит новую таксономию
4. Готово — никаких изменений в коде не требуется
